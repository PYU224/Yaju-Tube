# Version 1.~ 

## 1.8
- 再生画面がバックグラウンドに移行した際の挙動を修正
<br>別の画面へ移動した時（下部タブバーでの画面切り替えを含む）やアプリ自体をバックグラウンドにした時に、再生を自動停止してその時点の再生位置を履歴へ保存するようになりました。今までは下部タブバーで別画面に切り替えると音声が裏で流れ続け、しかも操作もできなくなることがありました
- 続きから再生（視聴履歴からの再開）が機能しない問題を修正
<br>埋め込みプレイヤーAPIの仕様変更に追従して再生位置を保存できるようにした上で、再生開始前は効かない seek に頼らず、埋め込みURLの start パラメータで開始位置を指定して確実に頭出しするようにしました。あわせて公式アプリと同じ基準（30秒未満の短い動画や、残り10秒以内まで視聴済みの動画は先頭から再生）で続きから再生するか判定します
